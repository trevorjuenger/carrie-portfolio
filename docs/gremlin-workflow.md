# Website Gremlin Workflow

Carrie should eventually be able to ask for things like:

- Add a new photography project
- Update my bio
- Replace my CV PDF
- Add a Vimeo link to a video page
- Publish my site

## Initial workflow
1. Add content files and assets
2. Regenerate site pages
3. Preview locally or on GitHub Pages
4. Publish changes

## Design principle
Keep content structured enough that Mori can safely update it from chat, without hand-editing every page.
